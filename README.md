# 🌊 CloudWave

**Простой и красивый музыкальный плеер на основе SoundCloud**

CloudWave — это минималистичный веб-плеер, который позволяет искать и слушать музыку прямо с SoundCloud. Проект создан как пет-проект на чистом HTML, CSS и JavaScript без использования фреймворков.

## ✨ Основные возможности

- 🔍 Поиск треков по SoundCloud
- 🎵 Просмотр результатов в красивых карточках (обложка + название + артист)
- ▶️ Воспроизведение треков через официальный SoundCloud Widget
- ❤️ Добавление треков в «Избранное»
- 📂 Отдельная страница «Избранное» с сохранением через `localStorage`
- 📱 Адаптивный дизайн (мобильные устройства поддерживаются)
- 🌊 Волновой стиль оформления (CloudWave theme)

## 🛠 Технологии

- **HTML5**
- **CSS3** (Flexbox, Grid, анимации, кастомные свойства)
- **Vanilla JavaScript**
- **SoundCloud Widget API** (iframe)
- **localStorage** для хранения избранного

## 🚀 Как запустить проект

1. Скачай или клонируй репозиторий
2. Открой файл `index.html` в браузере
3. Для корректной работы SoundCloud Widget рекомендуется запускать проект через локальный сервер (Live Server в VS Code или аналог)

> **Важно:** SoundCloud Widget требует, чтобы сайт открывался по `http/https`, а не через `file://`. Используй локальный сервер.

## 🎯 Функционал

### Главная страница (`index.html`)
- Поиск треков
- Отображение результатов в виде карточек
- Клик по карточке → воспроизведение в нижнем плеере

### Плеер
- Play / Pause
- Прогресс-бар
- Отображение текущего трека (обложка, название, артист)

### Избранное
- Добавление/удаление треков сердечком ❤️
- Сохранение в `localStorage`
- Отдельная страница `favorites.html`
